# Shot Tracker 2.0 - Concept A: Medical Professional

## Design Philosophy
Clean, clinical interface inspired by Apple Health and professional medical software.
Trustworthy, precise, data-dense but readable.

## Color Palette
- Primary: Clinical Blue (#0066CC)
- Backgrounds: Clean whites and light grays
- Accents: Trust blue, success green, warning amber
- Text: Dark grays for hierarchy

## Key Features
- ✅ 121-peptide searchable database
- ✅ Daily dose tracking with compliance metrics
- ✅ Schedule management with cycles
- ✅ Inventory tracking with cost analysis
- ✅ Reconstitution calculator
- ✅ Trends and analytics
- ✅ Professional charts and graphs

## iPhone Optimization
- Optimized for iPhone 12-15 Pro Max
- Responsive 375px - 428px
- Native-feeling interactions
