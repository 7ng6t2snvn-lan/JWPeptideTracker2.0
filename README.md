# Gelato Formulator

Professional gelato and sorbet recipe calculator with precise nutritional analysis.

## Features
- Recipe formulation with PAC/POD calculations
- Nutritional breakdown (fat, sugar, solids)
- Professional ingredient database
- Batch scaling
- PWA-enabled for offline use

## Latest Updates

- New ingredients: Cranberry Purée, Lemon Zest
- 15 total recipes

## Installation
1. Clone or download this repository
2. Open `index.html` in a web browser
3. For mobile: Add to home screen for PWA experience
